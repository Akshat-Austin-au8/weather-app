## Requirements
For development, you will only need Node.js and a node global package installed in your environement.
---
## Project Installation
```
$ git clone https://github.com/Akshat-Austin-au8/weather-app.git
$ cd weather-app
```
**Run npm install**
```
$ npm i
```
---

## Running The Project
```sh
   $ npm start
```
